Key Recovery Tool

As of early April 2017, Airbitz has replaced their proof-of-concept Edge Login system with a production-ready system. This means that private keys stored in the proof-of-concept system are no longer accessible through the Augur app directly. If you have accidentally sent real Ether or Rep to a private key stored in the old system, this tool will allow you to retrieve it.

To retrieve keys stored in the current system, please go to the Augur app and use the "Download Account Key File" feature instead.
